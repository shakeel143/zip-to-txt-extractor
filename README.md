🛠️ How It Works
ZIP file(s)
     │
     ▼
Browser (JSZip)
     │
     ├── Filter → text/code files only
     ├── Sort → clean structure
     ├── Strip → remove root folder (optional)
     │
     ▼
Formatted TXT file
